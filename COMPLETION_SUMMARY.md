## 🚀 How to Use

### Development
```bash
# Install dependencies
bun install

# Run dev server
bun run dev

# Open browser
http://localhost:5173
```

### Testing
```bash
# Run all tests
bun run test

# Run tests with UI
bun run test:ui

# Run tests in headed mode
bun run test:headed

# Show test report
bun run test:report
```

### Production
```bash
# Build for production
bun run build

# Preview production build
bun run preview
```
